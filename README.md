# Nghia Truong - CV

A professional academic CV built with LaTeX using the [AltaCV](https://github.com/liantze/AltaCV) document class.

## Building

Compile with pdfLaTeX:

```bash
pdflatex CV.tex
```

## Files

- `CV.tex` - Main CV document
- `CV.bib` - BibTeX bibliography for publications
- `altacv.cls` - AltaCV document class (v1.7.4)
- `CV.xmpdata` - PDF/A metadata

## Requirements

- LaTeX distribution (MiKTeX or TeX Live)
- AltaCV dependencies: `academicons`, `fontawesome5`, `tikz`, `pdfx`
- The `silence` package for warning suppression
